# 第一章 走进Java

## 1.Java技术体系

* 1.1 JDK 组成
    > Java程序设计语言
    > Java虚拟机
    > Java API类库

* 1.2 JRE 组成
    > Java API类库中的Java SE API子集
    > Java虚拟机

* 1.3 Java 技术体系的四个平台
    > Java Card ：小内存设备(如智能卡)上的平台
    > Java ME ：移动终端(如手机)上的平台
    > Java SE : 桌面级应用(如windows下的应用程序)，提供了完整的Java核心API，以前称为J2SE
    > Java EE : 支持使用多层架构的企业级应用(如ERP，CRM应用)，以前称为J2EE
